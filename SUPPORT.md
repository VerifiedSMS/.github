# Getting Support for VerifiedSMS

Welcome to VerifiedSMS! We're here to help you succeed with our SDKs and API.

## 📖 Documentation

Start with our official documentation:  
👉 [https://docs.verifiedsms.com](https://docs.verifiedsms.com)

It covers:
- Getting started guides
- API reference
- Authentication
- Webhook setup
- Best practices

## 🐛 Report a Bug

If you've found a bug in one of our SDKs, please open an issue in the **specific SDK repository** on GitHub.  
Use the issue template and provide as much detail as possible.

## 💡 Feature Requests

We welcome ideas! Open an issue with the `enhancement` label in the relevant SDK repository.

## 🔒 Security Vulnerabilities

**Do not use public issues.**  
Please email [security@verifiedsms.com](mailto:security@verifiedsms.com) – see [SECURITY.md](SECURITY.md) for details.

## 💬 General Questions

For questions about:
- Account setup or billing
- Integration advice
- SDK usage

Please email: [support@verifiedsms.com](mailto:support@verifiedsms.com)

We aim to respond within 1 business day.
